# NEXA IMMOBILIER - Plateforme d'Intermédiation

Bienvenue sur le dépôt officiel de **NEXA IMMOBILIER**.
Ce site permet la vente de terrains, maisons et véhicules avec un système d'intermédiation forcée.

## Fonctionnalités
- **Vitrines Clients :** Affichage des biens validés uniquement.
- **Intermédiation :** Tous les boutons de contact mènent vers l'administrateur (Babsy).
- **Dépôt Vendeur :** Formulaire de soumission avec sécurité anti-numéro de téléphone.
- **Back-office Admin :** Validation manuelle des annonces via Firebase.

## Installation
1. Configurer un projet sur **Firebase Console**.
2. Activer **Firestore Database** en mode test.
3. Remplacer le `firebaseConfig` dans les fichiers HTML.
4. Héberger sur **GitHub Pages**.
